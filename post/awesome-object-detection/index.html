Papers

R-CNN Series

2015

  • Faster R-CNN[Code_Matlab][Code_Python][Code_Tensorflow]
    • Shaoqing Ren, Kaiming He, Ross Girshick, Jian Sun. Faster R-CNN: Towards Real-Time Object Detection with Region Proposal Networks, TPAMI, 2016. (An older version appeared in NIPS 2015)
    • Summary: A Region Proposal Network (RPN).
    • Code note: The python version is mostly used.
  • Fast R-CNN[Code_Python][Code_Matlab]
    • Ross Girshick. Fast R-CNN, ICCV, 2015.
    • Summary: The basic network architecture for later R-CNN based researches.
  • Multibox_v2 [Code]
    • Christian Szegedy, Scott Reed, Dumitru Erhan, Dragomir Anguelov, Sergey Ioffe. Scalable High Quality Object Detection, CVPR, 2015.
    • Summary: The basic idea is similar to RPN

2014

  • R-CNN [Code]
    • Ross Girshick, Jeff Donahue, Trevor Darrell, Jitendra Malik. Rich feature hierarchies for accurate object detection and semantic segmentation, CVPR, 2014.
    • Summary: The first time object detection combined with deep learning.
  • Multibox_v1 [Code]
    • Dumitru Erhan, Christian Szegedy, Alexander Toshev, Dragomir Anguelov. Scalable Object Detection using Deep Neural Networks, CVPR, 2014.
    • Summary: This paper tought us that we can generate proposals by using a deep neural network.
